# ⛩️ ヤオヨロクリッカー (Yaoyoro Clicker)

> **「ただひたすらにインフレする数値を眺め、神々を観測せよ。」**

ブラウザで遊べる、和風ダークファンタジーの世界観を纏った、インフレ系クリッカーゲームです。  
現世に漂う「霊貨」を集め、八百万の神々を呼び寄せ（召集）、その神威によってさらなる効率を目指します。

## ✨ 主な機能 (Features)

- **無限のインフレ体験**: 効率（P）と倍率（M）、さらに「深淵倍率（E）」が織りなす圧倒的な数値上昇。
- **460柱の神々・妖怪**: `cards.js` に刻まれた膨大な八百万のデータ。
- **10億連ガチャ**: 神結晶（レア通貨）を使い、神々を全力で招集する快感。
- **レスポンシブ対応**: スマホでは全画面アプリのように、PCでは広大な広告スペースと共に快適に動作。
- **実績システム (Upcoming)**: 1000種類に及ぶニッチで遊び心満載のアチーブメント。
- **データ解析**: Amplitudeによる高度なユーザー行動分析の導入。
